# 🌦️ Weather Dashboard Pro

A modern, portfolio-grade weather dashboard built using Streamlit and OpenWeatherMap API.
Designed with a clean UI, dynamic themes, and real-time data visualization.

---

## ✨ Features

* 🌍 Auto-detect user location
* 🔍 Search weather by city
* 🌡️ Real-time temperature, humidity, wind, pressure
* 📊 24-hour temperature trend (chart)
* 📅 5-day forecast cards
* 🎨 Dark/Light mode toggle
* 🌈 Dynamic background based on weather
* ⚡ Fast, responsive UI

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Matplotlib
* OpenWeatherMap API

## ⚙️ Setup Instructions

### 1. Clone the repository

git clone https://github.com/YOUR_USERNAME/weather-dashboard-pro.git

### 2. Navigate into the project

cd weather-dashboard-pro

### 3. Install dependencies

pip install -r requirements.txt

### 4. Add your API Key

Update config.py:
API_KEY = "your_api_key_here"

Or set environment variable:
export API_KEY=your_api_key

### 5. Run the app

streamlit run app.py
